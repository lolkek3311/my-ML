## 🔧 Если возникли проблемы с установкой:

### Вариант 1: Обновить pip сначала
```bash
python -m pip install --upgrade pip
pip install -r requirements.txt
```

### Вариант 2: Установить без указания версий (если проблемы с совместимостью)
```bash
pip install pandas numpy matplotlib seaborn scikit-learn lightgbm hdbscan
```

### Вариант 3: Использовать conda (если есть Anaconda/Miniconda)
```bash
conda install pandas numpy matplotlib seaborn scikit-learn lightgbm
pip install hdbscan
```

### Вариант 4: Для Linux (Ubuntu/Debian)
```bash
# Сначала установите системные зависимости
sudo apt-get update
sudo apt-get install python3-dev python3-pip
pip install -r requirements.txt
```

## 💡 Примечания

- **scikit-learn** включает большинство нужных компонентов:
  - `sklearn.cluster`: KMeans, DBSCAN
  - `sklearn.preprocessing`: StandardScaler, LabelEncoder
  - `sklearn.model_selection`: train_test_split
  - `sklearn.ensemble`: RandomForestClassifier, IsolationForest
  - `sklearn.decomposition`: PCA
  - `sklearn.metrics`: classification_report, confusion_matrix, silhouette_score

- **os**, **glob**, **warnings** — встроены в Python, устанавливать не нужно

- Версии в requirements.txt актуальны на апрель 2026
