# Сканирование уязвимостей (Vulnerability Scanning)

## 1. Nikto - Веб-сканер уязвимостей

### Базовое использование
```bash
nikto -h https://example.com
nikto -h https://example.com -o output.txt
nikto -h https://example.com -Format html -o output.html
```

### Расширенные опции
```bash
# С указанием порта
nikto -h example.com -p 8080

# С использованием прокси
nikto -h example.com -useproxy http://127.0.0.1:8080

# С SSL
nikto -h example.com -ssl

# С конкретными плагинами
nikto -h example.com -Plugins "apacheusers,cookies,dictionary"

# Обновление плагинов
nikto -update

# Проверка нескольких хостов
nikto -h targets.txt
```

## 2. OWASP ZAP - Автоматизированный сканер

### CLI режим
```bash
# Запуск в режиме daemon
zaproxy -daemon -port 8090

# Автоматическое сканирование
zaproxy -quickurl https://example.com
zaproxy -quickurl https://example.com -quickout output.html

# Сканирование через API
curl "http://localhost:8090/JSON/ascan/action/scan/?url=https://example.com&recurse=true&inScopeOnly=false"
```

### Python API
```python
from zapv2 import ZAPv2

zap = ZAPv2(apikey='your_api_key', proxies={'http': 'http://127.0.0.1:8080'})

# Сканирование
zap.urlopen('https://example.com')
zap.spider.scan('https://example.com')
zap.ascan.scan('https://example.com')

# Получение результатов
print(zap.alert_generator.alerts())
```

## 3. Nuclei - Шаблонный сканер

### Базовое использование
```bash
# Сканирование одной цели
nuclei -u https://example.com

# Сканирование списка целей
nuclei -l targets.txt

# С конкретными шаблонами
nuclei -u https://example.com -t cves/
nuclei -u https://example.com -t vulnerabilities/
nuclei -u https://example.com -t exposures/

# По тегам
nuclei -u https://example.com -tags cve
nuclei -u https://example.com -tags xss,sqli
nuclei -u https://example.com -tags lfi,rfi

# Скорость
nuclei -u https://example.com -rate-limit 10
nuclei -u https://example.com -c 50

# Вывод
nuclei -u https://example.com -o results.txt
nuclei -u https://example.com -json -o results.json
```

### Установка шаблонов
```bash
nuclei -update-templates
nuclei -tl                            # Список шаблонов
nuclei -u https://example.com -s low  # По severity
```

### Кастомные шаблоны
```yaml
id: custom-vulnerability

info:
  name: Custom Vulnerability Check
  author: yourname
  severity: high
  tags: custom

requests:
  - method: GET
    path:
      - "{{BaseURL}}/vulnerable-endpoint"
    matchers:
      - type: word
        words:
          - "vulnerable"
```

## 4. SQLmap - SQL Injection

### Базовое использование
```bash
# Проверка URL
sqlmap -u "https://example.com/page?id=1"

# С параметрами POST
sqlmap -u "https://example.com/login" --data="username=admin&password=test"

# С куки
sqlmap -u "https://example.com/page?id=1" --cookie="session=abc123"

# С заголовками
sqlmap -u "https://example.com/page?id=1" --headers="X-Forwarded-For: 127.0.0.1"

# Уровень и риск
sqlmap -u "https://example.com/page?id=1" --level=5 --risk=3
```

### Определение БД
```bash
# Получение баннера
sqlmap -u "https://example.com/page?id=1" --banner

# Получение пользователей
sqlmap -u "https://example.com/page?id=1" --users

# Получение баз данных
sqlmap -u "https://example.com/page?id=1" --dbs

# Получение таблиц
sqlmap -u "https://example.com/page?id=1" -D database_name --tables

# Получение колонок
sqlmap -u "https://example.com/page?id=1" -D database_name -T table_name --columns

# Дамп данных
sqlmap -u "https://example.com/page?id=1" -D database_name -T table_name --dump
sqlmap -u "https://example.com/page?id=1" --dump-all
```

### Эксплуатация
```bash
# Обход WAF
sqlmap -u "https://example.com/page?id=1" --tamper=space2comment
sqlmap -u "https://example.com/page?id=1" --tamper=space2comment,between,randomcase

# SQL shell
sqlmap -u "https://example.com/page?id=1" --sql-shell

# OS shell
sqlmap -u "https://example.com/page?id=1" --os-shell

# Файловая система
sqlmap -u "https://example.com/page?id=1" --file-read=/etc/passwd
sqlmap -u "https://example.com/page?id=1" --file-write=shell.php --file-dest=/var/www/html/
```

### Tamper скрипты
```bash
# Список tamper скриптов
sqlmap --list-tampers

# Популярные tamper скрипты
sqlmap -u "URL" --tamper=space2comment          # Пробелы -> комментарии
sqlmap -u "URL" --tamper=between               # > -> BETWEEN
sqlmap -u "URL" --tamper=randomcase            # Рандомный регистр
sqlmap -u "URL" --tamper=charcode              # CHAR кодирование
sqlmap -u "URL" --tamper=base64encode          # Base64
sqlmap -u "URL" --tamper=apostrophemask        # Кавычки -> UTF-8
```

### Техники
```bash
# Boolean-based blind
sqlmap -u "URL" --technique=B

# Time-based blind
sqlmap -u "URL" --technique=T

# Error-based
sqlmap -u "URL" --technique=E

# UNION query
sqlmap -u "URL" --technique=U

# Stacked queries
sqlmap -u "URL" --technique=S
```

## 10. Автоматизация и отчёты

### Скрипт полного сканирования
```bash
#!/bin/bash
TARGET=$1
OUTPUT="scan_results_$(date +%Y%m%d)"

mkdir -p $OUTPUT

echo "[*] Starting Nikto scan..."
nikto -h $TARGET -Format html -o $OUTPUT/nikto.html

echo "[*] Starting Nmap scan..."
nmap -sS -sV -sC -O -oA $OUTPUT/nmap $TARGET

echo "[*] Starting Nuclei scan..."
nuclei -u $TARGET -o $OUTPUT/nuclei.txt -json -o $OUTPUT/nuclei.json

echo "[*] Starting SQLmap scan..."
sqlmap -u "$TARGET" --batch -o $OUTPUT/sqlmap

echo "[*] Scan complete! Results in $OUTPUT/"
```

### Скрипт для нескольких целей
```bash
#!/bin/bash
while IFS= read -r target; do
    echo "Scanning: $target"
    nikto -h $target -o "nikto_$(echo $target | tr '/' '_').txt"
    nuclei -u $target -o "nuclei_$(echo $target | tr '/' '_').txt"
done < targets.txt
```
